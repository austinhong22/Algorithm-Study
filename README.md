## 📌 참여 방법

1. 이 저장소를 `Fork` 합니다.
2. 생성된 원격 저장소에 `github ID`로 폴더를 생성합니다.
3. 생성된 폴더에 자신의 코드를 작성합니다. `🔥 주석을 습관화 합시다!`
4. 작성한 코드를 커밋 규칙을 준수하여 `commit`합니다.
5. 원본 저장소로 `Pull Request`합니다.
6. 다른 참여자들의 `Pull Request`를 보고 `코드 리뷰`를 진행합니다.

- Commit Convention
